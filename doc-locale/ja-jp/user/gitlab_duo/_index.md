GitLab Duoは、ソフトウェア開発ライフサイクルのあらゆる段階でAIネイティブな機能を提供します。

GitLab Duoには、エージェント型の機能（自律的に動作するエージェント）と非エージェント型の機能（特定のタスクに対するAIアシスタンス）が含まれています。

GitLab Duoの機能は、[IDE拡張機能](../../editor_extensions/_index.md)とGitLab UIから利用できます。
